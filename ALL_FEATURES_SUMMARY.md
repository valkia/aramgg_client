## 📈 性能评估

### 系统需求

| 配置 | 推荐 | 最低 |
|------|------|------|
| CPU | i5-9400 | i5-6400 |
| 内存 | 16GB | 8GB |
| 显卡 | GTX 1060 | GTX 750 |
| 硬盘 | SSD 256GB | HDD 100GB |

### 性能等级评估

```
优秀 (绿色):
  • 平均耗时 < 100ms
  • 内存占用 < 200MB
  • 对游戏基本无影响

良好 (橙色):
  • 平均耗时 < 200ms
  • 内存占用 < 300MB
  • 对游戏影响很小

一般 (深橙):
  • 平均耗时 < 500ms
  • 内存占用 < 500MB
  • 可能有轻微影响

较差 (红色):
  • 平均耗时 > 500ms
  • 内存占用 > 500MB
  • 可能影响游戏性能
```

## 📞 问题排查

### 问题: 性能评级为"较差"

**解决方案**:
1. 增加截图间隔（改为10秒或以上）
2. 减少保留截图数量
3. 关闭其他后台程序
4. 升级硬件配置

### 问题: 定时截图没有启动

**解决方案**:
1. 检查浏览器开发者工具的 Console
2. 确认 IPC 通信正常
3. 重启应用

### 问题: 浮窗没有显示

**解决方案**:
1. 确认 F1 截图成功（查看控制台）
2. 检查 WinrateOverlay 组件是否加载
3. 查看 z-index 是否被其他元素遮挡
