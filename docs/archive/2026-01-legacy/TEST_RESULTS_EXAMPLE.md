## 如何进行真实游戏测试

### 建议流程

1. **第一次测试**
   - 进一次游戏，进入海克斯选择
   - 手动按 F1 截图 5-10 次
   - 运行 `npm run test:screenshots`
   - 查看识别效果

2. **第二次测试**
   - 用自动截图功能进行 1-2 局游戏
   - 自动保存会有数十张截图
   - 运行 `npm run test:screenshots`
   - 收集足够的数据

3. **微调（如果需要）**
   - 根据结果调整颜色范围
   - 修改验证参数
   - 重新 `npm run build`
   - 再次测试

4. **验收**
   - 确认识别率 > 80%
   - 确认误检率 < 5%
   - 确认实时游戏中工作正常

